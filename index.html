What is BOAT UI?
BOAT UI is a production-ready, ultra-lightweight semantic HTML/CSS component library. Zero JavaScript, zero dependencies ‚Äî just drop in the CSS and go.
How do I install it?
Copy the files to your project and add <link rel="stylesheet" href="index.css"> to your HTML. That's it ‚Äî no npm, no build step.
Does it support dark mode?
Yes. Override the CSS custom properties in tokens.css to create any theme. All semantic color tokens update automatically across every component.
